# Web Traffic Analytics

## Objective

Analyze website traffic to understand user behavior and identify opportunities to improve engagement.

## Tools Used

- Python
- Pandas

## Metrics Analyzed

- Page Views
- Users
- Session Duration
- Bounce Rate

## Findings

- Home page receives the highest number of visitors.
- Product pages have longer session durations.
- Contact page has the highest bounce rate.
- Users spend more time on product-related content.

## Suggestions

- Improve the Contact page by adding FAQs or clearer calls to action.
- Optimize the Home page to direct visitors toward Products.
- Continue publishing engaging content to increase user retention.
