# Customer Churn Analysis

## Objective

Analyze customer data to identify why customers cancel their subscriptions.

## Tools Used

- Python
- Pandas

## Steps

1. Loaded the dataset.
2. Checked for missing values.
3. Counted churned and retained customers.
4. Compared monthly charges.
5. Analyzed contract types.

## Findings

- Customers with month-to-month contracts churn more.
- Customers with higher monthly charges have a higher churn rate.
- Long-term contracts improve customer retention.

## Conclusion

Customer retention can be improved by encouraging long-term contracts and offering incentives to high-risk customers.
