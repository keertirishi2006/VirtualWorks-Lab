# Interactive Dashboard Development

## Objective

Create a simple dashboard that presents business performance using charts.

## Tools Used

- Python
- Pandas
- Matplotlib

## Dashboard Metrics

- Monthly Sales
- Monthly Profit
- Customer Growth

## Insights

- Sales increased steadily throughout the year.
- Profit followed the sales trend.
- Customer count grew consistently.

## Conclusion

The dashboard provides a quick overview of business performance and helps identify sales and customer trends.
