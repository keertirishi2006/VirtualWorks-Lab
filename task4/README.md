# Geospatial Data Analysis

## Objective

Analyze sales data from different cities to identify locations with high business potential.

## Tools Used

- Python
- Pandas

## Dataset

The dataset contains:

- City
- State
- Latitude
- Longitude
- Sales

## Analysis

- Total Sales
- Highest Sales City
- Average Sales
- Top Performing Cities

## Findings

- Mumbai generated the highest sales.
- Delhi and Bengaluru also showed strong performance.
- Kochi and Coimbatore had comparatively lower sales, indicating opportunities for targeted business strategies.

## Conclusion

Businesses can use geographical sales analysis to identify high-performing regions and make informed expansion decisions.
